# 我写了个相册 App，专门用来找表情包

手机里存了上万张截图和表情包，每次翻半天找不到想要的那张，微信自带的搜索又不靠谱。于是自己动手写了一个。

## 它能干嘛

**AI 自动打标签**

接上你自己的第三方 API（GPT-4V、Claude、Gemini 都兼容），它会扫描相册，调用视觉模型识别每张图的内容，生成标签存到本地。比如一张熊猫头表情包，它会打上"黑白熊猫、愤怒表情、文字'你完了'"之类的标签。

因为用的是你自己的 API Key，图片不会经过我的服务器，标签全在本地 SQLite 里，没网也能用。

**自然语言搜索**

打完标签之后就可以用日常语言搜了。搜"猫翻白眼"、"熊猫头锤人"、"裂开 emoji"都能找到对应的图。支持多组组合搜索：逗号/空格表示"且"，句号表示"或"。比如"猫，橘。奶龙，黄"就是找（橘猫）或（黄奶龙）。

**临时分享**

这个是我最近加的功能，解决一个很实际的痛点：搜到表情包之后要发微信，但微信只能从文件夹里选图，你得记住它本来在哪个文件夹再去翻。

现在搜到或浏览到想要的表情包，多选之后点一下"临时分享"，它会自动把图片复制到系统 Camera 文件夹。因为新文件日期最新，打开微信选图时直接在最前面就能看到。到设定时间后自动删除副本，退出 App 也会清理，完全不留痕迹。

时间可以在设置里调，默认 10 秒，够你切到微信选图发送了。

**其他功能**

- 按文件夹浏览，支持缩放动画进出
- 图片多选：复制、剪切、粘贴、删除、分享、批量云端解析
- 快速打标签：输入标签后点击图片即可批量写入
- 文件夹隐藏/显示、文件移动自动追踪

## 怎么用

去 [GitHub Releases](https://github.com/qingkongfeixing/smart-album/releases) 下载 APK 安装就行。打开后先在设置里配置一个视觉模型的 API（OpenAI 兼容的都行），然后点扫描相册，剩下的就等着 AI 给你打工了。

## 技术相关

Flutter + SQFlite + Provider，Android 端用 MediaStore 读相册和媒体索引。代码全开源的，有兴趣的可以看看。
